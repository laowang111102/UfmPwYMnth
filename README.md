# Python简易学生信息管理系统 python579

## 项目概述

本项目是一个基于Python开发的简易学生信息管理系统，主要用于管理学生的基本信息，支持数据的添加、编辑和删除。

## 技术栈

- 后端：Python
- 数据库：MySQL

## 功能模块

- 学生信息添加
- 学生信息编辑
- 学生信息删除
- 学生信息查询

## 系统角色

- 管理员：负责学生信息的管理操作

## 运行环境

- 操作系统：兼容Windows、Linux和MacOS
- Python版本：3.x
- MySQL版本：5.x或更高

## 部署步骤

1. 安装Python环境
2. 安装MySQL数据库，并创建相应数据库和表
3. 导入项目代码
4. 安装所需依赖（例如：使用pip安装mysql-connector-python）
5. 运行程序

## 目录结构

```
.
├── main.py               # 程序主入口
├── models.py             # 数据模型定义
├── views.py              # 视图逻辑处理
└── utils.py              # 辅助功能模块
```
